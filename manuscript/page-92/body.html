Certainly this is dewanee, the madness. Do they never go to sleep? Now there is a cloud coming to cover that moon. If it were only a big enough cloud I might try to run away in the darkness. But I am tired.”

That same cloud was being watched by two good friends in the ruined ditch below the city wall, for Bagheera and Kaa, knowing well how dangerous the Monkey-People were in large numbers, did not wish to run any risks.
